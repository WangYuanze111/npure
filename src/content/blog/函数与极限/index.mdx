# 函数知识复习
任何函数都可以表示为一个奇函数和一个偶函数之和的形式

**$$f(x) = \frac{f(x) + f(-x)}{2} + \frac{f(x) - f(-x)}{2}$$**

其中，$\frac{f(x) + f(-x)}{2}$ 是偶函数，$\frac{f(x) - f(-x)}{2}$ 是奇函数。

基本初等函数包括以下几类：
1. **幂函数**：$f(x) = x^n$，其中$n$ 是一个实数。
2. **指数函数**：$f(x) = a^x$，其中$a > 0$ 且 $a \neq 1$。
3. **对数函数**：$f(x) = \log_a x$，其中$a > 0$ 且 $a \neq 1$。
4. **三角函数**：如 $\sin x$、$\cos x$、$\tan x$ 等。
5. **反三角函数**：如 $\arcsin x$、$\arccos x$、$\arctan x$ 等。

常用的三角函数性质：
1. $\sin^2 x + \cos^2 x = 1$
2. $\tan x = \frac{\sin x}{\cos x}$, 当 $\cos x \neq 0$ 时.
3. $1+\tan^2 x = \sec^2 x$，当 $\cos x \neq 0$ 时。
4. $1 + \cot^2 x = \csc^2 x$，当 $\sin x \neq 0$ 时。

双曲函数的定义及其反函数和导数：
1. 双曲正弦函数：$\sinh x = \frac{e^x - e^{-x}}{2}$，其反函数为 $\text{arsinh} x$，导数为 $\cosh x$。
2. 双曲余弦函数：$\cosh x = \frac{e^x + e^{-x}}{2}$，其反函数为 $\text{arcosh} x$，导数为 $\sinh x$。
3. 双曲正切函数：$\tanh x = \frac{\sinh x}{\cosh x}$，其反函数为 $\text{artanh} x$，导数为 $\text{sech}^2 x$。

其中，

$\text{arsinh} x = \ln(x + \sqrt{x^2 + 1})$

$\text{arcosh} x = \ln(x + \sqrt{x^2 - 1})$，当 $x \geq 1$ 时。

$\text{artanh} x = \frac{1}{2} \ln\left(\frac{1+x}{1-x}\right)$，当 $|x| < 1$ 时。
