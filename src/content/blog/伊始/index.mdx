# 想要写点什么
从刚上大一开始，就想写点什么，记录一下大学生活，记录一下自己的成长，记录一下自己的心路历程。可是每次想写的时候，总是觉得没有什么可写的，或者说没有什么值得写的。感觉自己过得很平淡，没有什么特别的经历，也没有什么特别的感受。
说到底还是缺乏执行力。我自己很喜欢看b站小约翰可汗的视频。小约翰制作了一个“硬核狠人”的系列节目，这些狠人出身不同、秉性各异，却都在不同的方面
整出了各种硬核狠活。如果说要寻找他们身上的共同点，恐怕唯一的答案便是**执行力超强** 。

现在是2026年3月份，大学的时光已经过了一半大多，每个人也都为自己未来而焦虑着、行动着。所以我开这个博客的初心，大概就是记录可能到来的考研备考时光，借此督促自己专心备考。
